Trouvez sur internet trois images :
une de pierre, une de feuille et l'autre de ciseaux ;

Créez dans votre page HTML une section qui contient ces trois images sur une même ligne ;

L'utilisateur va cliquer sur une image afin de faire son choix.
En utilisant des sélecteurs jQuery, faites-en sorte que lorsque l'utilisateur clique sur
une image, son choix s'affiche dans la console ;

Maintenant créez une div avec l'id userChoice, et remplacez le contenu de cette div par
le choix de l'utilisateur ;

Créez une div avec l'id computerChoice et y mettre dedans le choix de l'ordinateur
que vous avez obtenu suite à l'appel de la fonction getComputerChoice créée dans le cours
précédent ;

Ajoutez une div pour afficher le résultat, le texte doit être vert si vous avez gagné,
gris en cas d'égalité et rouge si vous avez perdu.
